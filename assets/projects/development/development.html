Overview

React is a popular open-source, front end, Javascript library for building user interfaces or UI components. As I learned more about UI design and prototyping, I grew more curious about implementing one using React. Hence, in this project, I created a simple user-friendly list interface in React with libraries such as react-bootstrap and material-ui.

Web application

I chose to create a web application named Artshop for users who are interested in buying paintings. In this interface, a user can pick a set of paintings based on their preference of size, shape, and price - all in one page.

Usability and Design Principles

The following principles were considered while creating the web application:

Visual Layout

  • Minimalistic: The home page focuses on what is important - browsing the paintings with user selected preferences. The application has clearly labeled action items, which allow users to view whatever they want to without confusion.
  • Distinguishable sections: Each section of the page is visibly distinguishable. Hence, the user is able to quickly understand the different purposes of each section of the page - namely, a nav bar, display of products, and a shopping cart.

Alignment

  • Grids and Cards: Each card, whether they are in the shopping cart or in the display section, are grouped in a responsive grid. This allows for easy scrolling and viewing of products in different screen sizes.

Typography

  • Font: The same font is used throughout the application for consistency.
  • Hierarchy: Similar text size signifies similar importance. The title of the application or section (such as the navbar type) have different text sizes to signify different purposes and importance.

Affordance

  • Card: Each card is similarly styled to better recall how each product is displayed.
  • Action Items: The navbars are grouped together for better navigation and selection amongst filters. In addition, every "add to cart" and other buttons in the aggregator section are consistent and familiar with other commonly used interfaces.

Takeaways

  • By creating a react application, I was able to better understand components and importance of structure and design from the perspective of coding the application.

  • The libraries commonly used with React are very useful to build and customize the primary components while allowing the interface to be a familiar one at the same time.
